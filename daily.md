- docker 指令

  - 查詢現在運行的 container
    `docker ps -a`

  - Remove container

    - Stop all containers
      `docker stop $(docker ps -a -q)`
    - Remove all containers
      `docker rm $(docker ps -a -q)`

  - 移除所有未使用的 Docker 項目
    `docker system prune`

  - remove 所有 image
    `docker rmi $(docker images -a -q)`
